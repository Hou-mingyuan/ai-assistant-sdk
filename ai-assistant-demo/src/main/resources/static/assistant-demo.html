带 Vue 助手的 Demo

点击右下角悬浮球打开助手 → 切换到摘要 → 在输入框输入一句足够长的说明（至少约 12 个字），例如：
请根据下方「Demo 页面正文」区块的内容，用要点列出本页在讲什么
发送后，助手会把本页 article.demo-assistant-page-context 内的文字一并送给模型（由 pageContextBlocks 配置）。

Demo 页面正文

本页用于演示 AI Assistant SDK 在纯 HTML 页面中的集成方式：通过 CDN 加载 Vue 3， 再加载与本仓库一并构建的 ai-assistant.umd.js，调用 app.use(AiAssistant.default, options) 并在 options 里声明 pageContextBlocks，即可把当前 DOM 中的说明性正文拼入用户消息，从而无需手动复制整页文字去做摘要。

项目中还包含仅使用 fetch 调用 REST 的 index.html，那种方式不会加载 Vue 挂件，因此无法实现「分析当前页 DOM」； 若你希望验证页面级摘要，应使用本页或在自己的 Vue 根组件中挂载 <AiAssistant />。

后端仍为 Spring Boot Starter，默认上下文路径为 /ai-assistant。 请确保已配置可用的 LLM api-key（见 application.yml），否则助手请求会失败。
